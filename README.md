# Void Conflict — Rogue Trader Naval Combat

A Foundry VTT module for **Dark Heresy 2e** with **Rogue Trader** void ship combat rules.

## Features
- **Void Ship Actor** — full ship character sheet with hull, shields, crew, morale, weapons, components, and crew
- **Automated combat actions** — Lock On, Evasive Manoeuvres, Broadside, Ram, Damage Control, all roll dice and post to chat
- **Void Battle Map** — tactical overlay showing all ships on a starfield grid
- **Gothic aesthetic** — styled to match the Rogue Trader videogame

## Installation
Paste this URL into Foundry → Add-on Modules → Install Module → Manifest URL:
```
https://raw.githubusercontent.com/YOURUSERNAME/void-conflict/main/module.json
```

## How to Use
1. Enable the module in your world (Settings → Manage Modules)
2. Create a new Actor → choose type **"Void Ship"**
3. Fill in your ship stats on the sheet
4. Click **"Open Void Battle Map"** to start void combat
5. Or run macro: `VoidConflict.openBattle()`
